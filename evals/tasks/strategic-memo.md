# Eval Task — Strategic Memo

## Task
Write a short strategic memo arguing whether Humanprint should add semantic retrieval now or delay until the corpus reaches 80-150 examples.

## Reader
Maintainers deciding the build sequence.

## Required pattern coverage
- `PATTERN_STRATEGY_001`
- `PATTERN_PROOF_001`

## Must demonstrate
- Decision, principle, tradeoff, refusal, and next step.
- Evidence from current corpus size or roadmap sequence.
- Clear reason why pure vector search can be vibes-based too early.

## Failure modes
- Balanced essay that avoids a decision.
- "Semantic retrieval is powerful" without sequencing logic.
- No operating consequence.
