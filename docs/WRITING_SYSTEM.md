# Humanprint Writing System

## Prime directive

Write like a specific human speaking to one specific reader about one specific thing that matters.

If the prose could appear in a generic SaaS blog, it fails.

## The CLEAR foundation

- **Cut ruthlessly**: remove anything that does no work.
- **Lead with the point**: first sentence carries the promise, conflict, or conclusion.
- **Eliminate dead language**: no clichés, filler, corporate fog, or AI cadence.
- **Active construction**: name who did what.
- **Read aloud**: if the mouth stumbles, the mind stumbles.

## The anti-slop standard

Bad AI writing is usually not wrong. It is worse: it is frictionless.

It sounds clean but has no fingerprints. To fix that, force:

1. **A real reader**: Who exactly is this for?
2. **A real stakes**: What changes if they believe this?
3. **A real claim**: What are we saying that can be argued with?
4. **Real proof**: What detail makes it undeniable?
5. **A real voice**: What would only this author say this way?

## Drafting protocol

1. **Brief**: define reader, goal, awareness stage, proof, voice, forbidden phrases.
2. **Angle**: choose the sharpest promise or tension.
3. **Hook sprint**: write 10-20 hooks before drafting.
4. **Draft**: write fast. Do not polish early.
5. **Cut pass**: remove 15-20% minimum.
6. **Proof pass**: add numbers, names, examples, contrasts.
7. **Voice pass**: add lived judgment, not decorative personality.
8. **Read-aloud pass**: rewrite every stumble.

## Persuasion matching

Use the reader's awareness stage before choosing structure:

- **Unaware**: story or surprising problem.
- **Problem-aware**: PAS — problem, agitate, solve.
- **Solution-aware**: differentiate mechanism.
- **Product-aware**: proof and objection handling.
- **Most-aware**: direct offer, urgency, friction removal.

## Good output test

The piece should make the reader think one of these:

- "I hadn't thought of it that way."
- "That is exactly my problem."
- "This person has actually done the work."
- "I want to steal that line."
- "Send this to someone."
