# Annotation Guide

Every Humanprint library item must teach a future agent how to write better.

## Required questions

For every example, answer:

1. What type of writing is this?
2. Why is it excellent?
3. What craft moves does it use?
4. When should an agent imitate it?
5. What should an agent avoid copying blindly?

## Annotation standard

Do not write vague praise like “strong writing” or “great hook.” Name the mechanism.

Weak:
> Good hook.

Strong:
> The hook creates tension by contradicting a common belief, then delays the explanation long enough to create forward motion.

## Required sections

Each example should include:

- Source metadata
- Rights note
- Why this is good
- Craft moves
- Structure map
- Short excerpt or summary
- Reusable pattern
- Imitation prompt
- Anti-patterns to avoid

## Rights policy in annotations

If the source is modern/copyrighted, store metadata and analysis. Use short excerpts only when necessary. Never store full copyrighted works without permission.

## Quality score

Score 1-10. A 9 or 10 should have at least one elite trait:

- unforgettable opening
- unusually clear explanation
- strong proof chain
- elegant structure
- memorable metaphor
- persuasion density
- strategic framing
- distinctive voice
- compression
