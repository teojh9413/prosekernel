# Anti-Slop Checklist

Run this before publishing any AI-assisted draft.

## Immediate fail phrases

Delete or rewrite if present:

- In today's fast-paced world
- Ever-evolving landscape
- Delve into
- Unlock / unleash
- Leverage
- Game-changer
- Seamless / robust / cutting-edge
- Revolutionary / transformative without proof
- It is important to note
- In conclusion
- This article explores

## Structural checks

- Does the first sentence contain the point, tension, proof, or scene?
- Can the reader understand the argument from the first sentence of each paragraph?
- Is there one idea per paragraph?
- Does every section move the reader forward?
- Is the ending earned instead of summarized?

## Specificity checks

- Replace abstractions with examples.
- Replace adjectives with proof.
- Replace "many" with a number or range.
- Replace "users" with a named persona.
- Replace "better" with the exact dimension: faster, cheaper, safer, easier, sharper.

## Voice checks

- Would a real person say this out loud?
- Is there a sentence only this author would write?
- Is there a concrete judgment or point of view?
- Did we remove fake enthusiasm?
- Did we avoid symmetrical AI lists unless structure is useful?

## Final cut

Cut 15-20%. If nothing hurts, you did not cut enough.
