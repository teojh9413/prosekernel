# Orwell: Politics and the English Language

## Source
- Author: George Orwell

## Why this is good
Orwell connects language quality to thought quality. Bad prose is not just ugly; it hides agency, dulls judgment, and lets writers avoid saying what they mean.

## Craft moves
- Attacks dead metaphors and prefabricated phrases.
- Makes editing moral, not cosmetic.
- Gives simple rules that can be applied immediately.
- Uses concrete bad examples to make vague failure visible.

## Structure map
1. Diagnosis: language decay and thought decay reinforce each other.
2. Examples: bad prose in the wild.
3. Mechanism: stale phrases replace thinking.
4. Rules: practical editing discipline.
5. Larger implication: clear language protects clear thought.

## Excerpt or summary
Humanprint takeaway: never use a phrase you did not choose consciously.

## Reusable pattern
When a sentence sounds familiar, interrogate it. If it came preassembled, rebuild it from meaning.

## Imitation prompt
Edit this draft using Orwell’s rules: cut stale metaphors, shorten words, prefer active voice, remove jargon, and preserve meaning over ornament.

## Anti-patterns to avoid
- Turning plain language into flat language.
- Cutting so much that nuance disappears.
- Using Orwell as an excuse for bluntness without precision.
