# Shopify Polaris actionable language

## Source
- Author: Shopify Polaris
- URL: https://polaris.shopify.com/content/actionable-language

## Why this is good
Polaris content guidance is strong because it treats copy as part of the interface, not decoration. Actionable language reduces uncertainty: users know what is happening, what they can do, and what the result will be.

## Craft moves
- Uses verbs tied to user actions.
- Connects labels to consequences.
- Reduces ambiguity in commerce workflows.
- Keeps copy short without becoming cryptic.

## Structure map
1. User situation.
2. Available action.
3. Consequence or result.
4. Recovery path if needed.

## Excerpt or summary
UX-writing guideline analysis only; no guide text mirrored.

## Reusable pattern
Interface copy should pair the action with the consequence the user cares about.

## Imitation prompt
Rewrite this interface copy so each label or message names the action, consequence, and next step clearly.

## Anti-patterns to avoid
- Cute labels that obscure function.
- Generic buttons detached from outcome.
- Short copy that creates uncertainty.
