## Why this is good
Mailchimp’s guide is valuable because it distinguishes brand voice from situational tone. Product copy should not sound equally playful during success, confusion, and failure. The craft move is contextual empathy.

## Craft moves
- Separates voice from tone.
- Adjusts language based on user state.
- Uses examples to show boundaries.
- Prevents brand personality from harming usability.

## Structure map
1. Stable voice principles.
2. User context/emotion.

## Reusable pattern
Good microcopy adapts tone to the user’s emotional state while preserving a consistent voice.

## Imitation prompt
Write product microcopy variants for success, confusion, and error states using one consistent voice but different tone.

## Anti-patterns to avoid
- Being playful when the user is stressed.
- Flattening all product states into the same voice.
- Using brand personality where clarity is needed.
