# Python tutorial appetite opening

- Author: Python Software Foundation

## Why this is good
The Python tutorial does not begin as a reference dump. Its appetite-setting opening gives the learner a reason to continue, places the language in practical context, and lowers the intimidation barrier before syntax begins.

## Craft moves
- Starts with motivation before mechanics.
- Frames the tool around what it lets the learner do.
- Uses a conversational threshold instead of assuming expert identity.
- Defers exhaustive detail until the reader has a reason to care.

## Structure map
1. Why this language is worth learning.
2. What kinds of problems it helps with.
3. What the tutorial will and will not cover.
4. Transition into first concrete concepts.

## Excerpt or summary

## Reusable pattern
Before a technical tutorial, write an appetite section that turns curiosity into enough motivation to tolerate syntax and setup.

## Imitation prompt
Open a technical tutorial by explaining the reader payoff, practical uses, and learning path before introducing commands or API details.

## Anti-patterns to avoid
- Starting with installation minutiae before motivation.
- Assuming the reader already knows why the topic matters.
- Turning the opening into a complete history lesson.
