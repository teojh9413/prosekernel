# Nike — Just Do It

## Source
- Author: Nike
- URL: https://about.nike.com/en/newsroom/releases/nike-why-do-it-campaign
- Rights note: Short phrase referenced for analysis.

## Why this is good
The line is almost all verb. It collapses hesitation, identity, and action into three words. It works because it does not explain the brand. It commands the reader into the brand’s worldview.

## Craft moves
- Uses imperative action.
- Removes excuses and explanation.
- Makes the brand feel like a challenge.
- Works across skill levels: beginner, amateur, elite.

## Structure map
1. Direct command.
2. Implied resistance.
3. Identity through action.
4. Open-ended applicability.

## Excerpt or summary
Short referenced line: “Just Do It.”

## Reusable pattern
For action brands: reduce the brand promise to the command the customer needs to hear at the moment of hesitation.

## Imitation prompt
Create 30 imperative positioning lines for this brand. Each should attack a real hesitation and invite action without explaining too much.

## Anti-patterns to avoid
- Using a command when the category requires trust or nuance.
- Mistaking shortness for power.
