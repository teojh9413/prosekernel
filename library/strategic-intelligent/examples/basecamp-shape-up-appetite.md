# Basecamp Shape Up — Appetite as Strategy

## Source
- Author: Basecamp
- URL: https://basecamp.com/shapeup
- Rights note: Metadata and analysis only.

## Why this is good
“Appetite” is a strategic writing move because it forces a decision before details swallow the discussion. Instead of asking “how long will this take?” it asks “how much is this worth?”

## Craft moves
- Reframes estimation as willingness to invest.
- Makes tradeoffs explicit early.
- Prevents scope from pretending to be inevitable.
- Connects strategy to time budget.

## Structure map
1. Desired outcome.
2. Appetite/time budget.
3. Shaped solution.
4. Constraints.
5. Cutline: what will not be done.

## Excerpt or summary
Summary: define the appetite for a project before shaping the solution so the team solves within a strategic boundary.

## Reusable pattern
“We want [outcome], but it is only worth [time/effort]. Therefore the solution must fit [constraint].”

## Imitation prompt
Turn this bloated project proposal into an appetite-driven strategy memo. Name the outcome, investment limit, tradeoffs, and no-gos.

## Anti-patterns to avoid
- Using “strategy” to avoid hard constraints.
- Letting every stakeholder add scope after appetite is set.
