# Mozilla Manifesto principles as operating commitments

## Source
- Author: Mozilla
- URL: https://www.mozilla.org/en-US/about/manifesto/
- Rights note: Metadata and original craft analysis only; do not mirror full source text.

## Why this is good
The Mozilla Manifesto is strategic writing because it states principles that imply decisions, alliances, and refusals. It is not only brand sentiment; it creates a public operating philosophy for how the organization should behave.

## Craft moves
- Turns values into public commitments.
- Names the system-level stakes, not only product benefits.
- Makes the organization accountable to principles.
- Creates a basis for future decisions and critique.

## Structure map
1. Define the shared public good.
2. State principles about that good.
3. Connect principles to organizational responsibility.
4. Invite participation and accountability.

Manifesto structure and principle analysis only; no source text mirrored.

## Reusable pattern
Write principles so they create commitments the organization can be judged against.

## Imitation prompt
Draft an organizational manifesto that names the shared good, states decision-grade principles, and invites accountability.

## Anti-patterns to avoid
- Writing values that cannot change decisions.
- Using universal virtue words without tradeoffs.
- Making the manifesto about the organization instead of the public stake.
