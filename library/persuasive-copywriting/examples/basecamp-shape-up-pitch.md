# Basecamp Shape Up — Write the Pitch

## Source
- Author: Basecamp
- URL: https://basecamp.com/shapeup/1.5-chapter-06
- Rights note: Metadata and craft analysis only.

## Why this is good
A Shape Up pitch persuades by making scope concrete. It does not sell a vague idea. It defines the problem, appetite, solution shape, rabbit holes, and no-gos so decision-makers can judge the bet.

## Craft moves
- Starts with problem, not feature enthusiasm.
- Defines appetite before solution sprawl.
- Names risks and exclusions upfront.
- Makes the proposal discussable.

## Structure map
1. Problem.
2. Appetite.
3. Solution shape.
4. Rabbit holes.
5. No-gos.
6. Decision.

## Excerpt or summary
Summary: good product pitches define the shape of a solution and its constraints before committing a team.

## Reusable pattern
“We are willing to spend [appetite] to solve [problem] by shaping [solution], while avoiding [rabbit holes/no-gos].”

## Imitation prompt
Write a Shape Up-style pitch for this idea. Include problem, appetite, solution sketch, rabbit holes, no-gos, and a clear bet.

## Anti-patterns to avoid
- Pitching a full spec instead of a shaped bet.
- Hiding uncertainty to sound confident.
