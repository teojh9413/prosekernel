# Halbert Starving Crowd Principle

## Source
- Author: Gary Halbert
- URL: https://www.thegaryhalbertletter.com/
- Rights note: Analysis only. Do not copy full letters.

## Why this is good
Halbert’s “starving crowd” principle forces copywriters to start with demand. The best copy cannot save an offer nobody wants. This makes writing more strategic: find the hunger before polishing the pitch.

## Craft moves
- Makes market selection part of writing.
- Connects desire to offer before style.
- Turns features into outcomes with “so that you can.”
- Handles objections conversationally.

## Structure map
1. Identify hungry reader.
2. Name pain/desire.
3. Show mechanism.
4. Translate feature to outcome.
5. Remove risk.
6. Ask directly.

## Excerpt or summary
The strongest marketing advantage is not product or copy; it is a market that already wants the thing.

## Reusable pattern
Feature → “so that you can” → concrete outcome.

## Imitation prompt
Rewrite this offer for a starving crowd. First define who already wants it badly, then write to that person only.

## Anti-patterns to avoid
- Emotional manipulation without truth.
- Fake scarcity.
- Clever copy for a cold market.
