# NN/g Inverted Pyramid for Social Attention

## Source
- Author: Nielsen Norman Group
- URL: https://www.nngroup.com/articles/inverted-pyramid/
- Rights note: Metadata and analysis only.

## Why this is good
The inverted pyramid is useful for social writing because it respects skim behavior. It says: put the most important point first because many readers will not reward a slow build.

## Craft moves
- Front-loads the conclusion.
- Gives impatient readers value immediately.
- Turns structure into reader empathy.
- Makes each later section increasingly optional.

## Structure map
1. Main point first.
2. Essential support second.
3. Useful context third.
4. Extra detail last.

## Excerpt or summary
Summary: web readers often skim; writing should begin with the conclusion or most important information.

## Reusable pattern
Lead with the point. Then earn the reader’s patience with proof, context, and nuance.

## Imitation prompt
Rewrite this thread using an inverted pyramid: first line gives the core claim, next lines give proof, final lines add nuance or optional detail.

## Anti-patterns to avoid
- Confusing “lead with conclusion” with spoiling all curiosity.
- Burying the emotional reason to keep reading.
