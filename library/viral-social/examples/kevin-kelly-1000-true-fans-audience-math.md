# Kevin Kelly 1000 True Fans audience math

## Why this is good
The idea travels because it converts creator anxiety into manageable arithmetic. Instead of asking for mass fame, the frame gives readers a smaller, repeatable target with a memorable number and a new definition of sufficiency.

## Craft moves
- Uses a specific number to make an abstract market feel graspable.
- Reframes success away from mass popularity.
- Gives the audience a phrase they can use in planning conversations.
- Makes the model emotionally relieving without pretending it is effortless.

## Structure map
1. Name the alternative model.
2. Define the core number and relationship.
3. Show why it changes the creator equation.
4. Discuss implications and constraints.

## Reusable pattern
Turn an intimidating scale problem into a memorable sufficiency threshold with simple math and a clear audience unit.

## Imitation prompt
Write a viral idea post that gives a large ambition a smaller numeric target, explains the unit, and shows why it changes behavior.

## Anti-patterns to avoid
- Using a number that is decorative rather than explanatory.
- Promising certainty from a simple model.
- Forgetting the constraints behind the appealing math.
