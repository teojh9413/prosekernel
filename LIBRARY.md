# Humanprint Library Catalog

This is the master catalog of writing examples and source notes.

Each entry is not merely a “good example.” It is a reusable teaching object for future agents.

## Current corpus

- Total annotated examples: 60
- Populated categories: 12
- Target categories scaffolded: 12 total top-level categories
- Rights posture: metadata/analysis-first, short excerpts only when useful and safe

## How to use this catalog

- Pick the category closest to the writing task.
- Study 3-5 examples before drafting.
- Transfer structure, not wording.
- Extract the craft move, then write a new piece for the current reader.

## Corpus depth principle

One example gives a direction. Three to five examples create a pattern. Ten examples create taste.

## Tooling

- `humanprint lint <file>` — score drafts for AI-slop markers.
- `humanprint new-example ...` — generate a compliant annotation skeleton.
- `humanprint validate-library` — validate example files for required sections.
- `humanprint search-examples "task"` — retrieve examples for a writing task.
- `humanprint write-demo "task"` — generate a deterministic retrieval + draft + lint report.

## Phase 4 collection targets

- Move the highest-priority public categories toward 10 examples each: persuasive/copywriting, technical/explanatory, viral/social, brand/positioning, email/newsletters, strategic/intelligent, UX/product microcopy, and crisis communications.
- Keep all additions rights-safe: metadata, source URLs, rights classification, original craft analysis, reusable patterns, and anti-patterns.
- Prefer examples that teach a repeatable move over examples that are merely famous.
