# Narrative Frames

## Problem → mechanism → proof → action

Best for persuasive writing.

## Scene → tension → insight → implication

Best for essays and social posts.

## Context → choice → tradeoff → principle

Best for strategic memos and founder letters.

## Claim → example → distinction → rule

Best for intelligent/explanatory writing.


## Shared value → contradiction → consequence

Best for moral, strategic, or brand arguments.

1. Name the shared value.
2. Show the contradiction.
3. Make the reader confront the consequence.
4. Offer the decision or action.

Source patterns: Frederick Douglass and MLK.

## Trend → why now → opportunity → invitation

Best for market theses and startup strategy.

Source pattern: YC Requests for Startups.
