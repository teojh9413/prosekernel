# Objection Handling Patterns

## Feel / Felt / Found

Use emotional safety before correction.

1. I know how you feel.
2. I felt the same way.
3. Here is what I found.

## Even-if proof

Template:
> This works even if [common objection], because [mechanism/proof].

## Risk reversal

Template:
> If [bad outcome], then [safety mechanism].

## Comparison frame

Template:
> Unlike [alternative], this works by [different mechanism].
